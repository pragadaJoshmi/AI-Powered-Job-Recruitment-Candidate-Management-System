# AI-Powered Job Recruitment & Candidate Management System (AIRCMS)

## Project Overview
The AI-Powered Job Recruitment & Candidate Management System (AIRCMS) is a Salesforce-based application developed to automate and streamline the recruitment process. The system manages job openings, candidate applications, interview tracking, offer approvals, and onboarding using Salesforce automation and Agentforce AI.

## Objectives
- Automate the recruitment process
- Manage job openings and candidate applications
- Improve hiring decisions using Agentforce AI
- Automate approvals and notifications
- Secure recruitment data using Profiles, Roles, and Users
- Reduce manual work using Salesforce automation

## Technologies Used
- Salesforce CRM
- Agentforce AI
- Lightning Experience
- Flow Builder
- Approval Process
- Validation Rules
- Reports & Dashboards

## Features
- Job Opening Management
- Candidate Application Management
- Automated Recruitment Workflow
- Approval Process Automation
- AI-Based Candidate Evaluation
- Reports and Dashboards
- Role-Based Security
